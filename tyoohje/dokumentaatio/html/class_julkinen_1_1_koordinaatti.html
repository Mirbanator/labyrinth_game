Yksityiskohtainen selite

Luokka pelin sijainti tietojen esittämiseen.

Koordinaatti luokka sisältää X- ja Y-koordinaattin ja tiedon siitä että onko kyseessä irtopala.

Koordinaatti -luokasta tehtyyn olioon ei voi sijoittaa uusia arvoja, muuten kuin toisen koordinaatin sijoittamalla.

Koordinaatti -luokasta tehdyt oliot tarjoavat sisältämiinsä arvoihin hae metodit, mutta arvoihin itseensä ei pääse suoraan käsiksi.

bool Julkinen::Koordinaatti::operator< ( Koordinaatti const &  koordinaatti) const

Pienemmyysvertailuoperaattori.

Vertaa kohde koordinaatin arvoja omiin arvoihinsa.

Jälkiehto
Palauttanut tiedon siitä onko olio "pienempi" kuin parametri. Pienemmyydessä vertaillaan ensin X- ja sitten Y-koordinaatteja, ja irtopala on kaikkein pienin. (Vertailu tarvitaan, jos Koordinaatteja haluaa käyttää esim. map:n avaimina.) Poikkeusturvallisuus: No-throw takuu.
Parametrit

Palauttaa
Palauttaa true jos olio on "pienempi" kuin parametri. Palautaa false muuten.
