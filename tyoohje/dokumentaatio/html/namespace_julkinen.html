Yksityiskohtainen selite

Nimiavaruus harjoitustyön valmiina annetulle koodille.

Kaikki harjoitustyön valmiina annettu koodi on nimiavaruudessa Julkinen. Ko. nimiavaruuteen ei ole suotavaa laittaa omaa koodia.

◆ Suunta

enum Julkinen::Suunta

Pelaajan liikumissuunnat.

Luettelotyyppien arvot
ALAS 

Pelaaja liikuu alaspäin.

AUTOMAATTI 

Pelaaja liikuu automaattisesti kohti seuraavaa palkintoa.

Pelaaja liikkuu ensisijaisesti siihen suuntaan missä matka palkintoon on pidempi, jos siihen suuntaan ei voi liikkua, niin pelaaja liikuu siihen suuntaa, missä matka palkintoon on lyhyempi. Jos pelaaja ei pysty liikumaan kumpaankaan suuntaan, niin pelaaja pysyy paikallaan.

Pelaaja ei ikinä liiku pois päin palkinnosta, paitsi jos hänen poimimansa esineet aiheuttavat moisen siirtymän.

OIKEALLE 

Pelaaja liikuu oikealle.

VASEMMALLE 

Pelaaja liikuu vasemmalle.

YLOS 

Pelaaja liikuu ylöspäin.

PAIKALLAAN 

Pelaaja pysyy paikallaan.

Funktioiden dokumentaatio

◆ luoPeli()

std::unique_ptr<Pelirajapinta> Julkinen::luoPeli ( )

Luo instanssi pelirajapinnasta.

Pääohjelma kutsuu tätä funktiota saadakseen itselleen instanssin oliosta, joka toteuttaa pelirajapinnan. Funktion toteutus on tehtävä omaan koodiin.

Jälkiehto
Luodun peli-instanssin tuhoamisvastuu siirtyi pääohjelmalle. Luotu olio on alustustilassa.
Palauttaa
Dynaamisesti luotu instanssi luokasta, joka toteuttaa Labyrintti-pelin.

◆ vaittamaToteutus()

bool  vaite,
char const *  lauseke,
char const *  tiedosto,
unsigned int  rivi,
char const *  funktio 
)

Väittämien tarkistamiseen käytettävä funktio.

Tätä funktiota ei ole tarkoitettu kutsuttavaksi suoraan. Käytä tätä ESIEHTO()-, INVARIANTTI()- tai JALKIEHTO()-makron avulla.

Esiehto
lauseke ≠ 0 ∧ tiedosto ≠ 0
Jälkiehto
Funktio ei koskaan palaa normaalisti. Perustakuu poikkeuksen sattuessa.
Parametrit
[in]tyyppiVäittämän tyyppi.
[in]vaiteTestatun väittämän totuusarvo.
[in]lausekeTestatun väittämän lauseke merkkijonona.
[in]tiedostoSen tiedoston nimi, jossa väittämä oli.
[in]riviRivinumero, jolla väittämä oli tiedostossa.
[in]funktioSen funktion nimi, jossa väittämä oli (jos saatavilla).
